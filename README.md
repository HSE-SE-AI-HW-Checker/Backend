# Простой HTTP сервер на FastAPI

Это простой HTTP сервер, созданный с использованием фреймворка FastAPI. Сервер предоставляет базовые эндпоинты для демонстрации возможностей FastAPI.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер одним из способов:

   **Способ 1 (рекомендуемый):**
   ```bash
   source venv/bin/activate && python run_server.py
   ```
   
   **Способ 2:**
   ```bash
   source venv/bin/activate && python main.py
   ```

Сервер будет доступен по адресу: http://localhost:8000

## Доступные эндпоинты

### Основные эндпоинты:
- `GET /` - Главная страница с HTML-интерфейсом
- `GET /hello` - Простое приветственное сообщение
- `GET /status` - Проверка статуса сервера
- `GET /info` - Информация о сервере

### Эндпоинты для работы с товарами:
- `GET /items` - Получить список всех товаров (с возможностью фильтрации по категории)
- `GET /items/{item_id}` - Получить товар по ID
- `POST /items` - Создать новый товар
- `PUT /items/{item_id}` - Обновить существующий товар
- `DELETE /items/{item_id}` - Удалить товар по ID

### Эндпоинты для работы с пользователями:
- `GET /users` - Получить список всех пользователей
- `GET /users/{user_id}` - Получить пользователя по ID
- `POST /users` - Создать нового пользователя

### Дополнительные эндпоинты:
- `GET /search?q={query}` - Поиск по товарам и пользователям
- `GET /health` - Проверка здоровья сервера
- `GET /stats` - Получить статистику по данным

## Примеры использования

### Получение списка товаров:
```bash
curl http://localhost:8000/items
```

### Фильтрация товаров по категории:
```bash
curl http://localhost:8000/items?category=Электроника
```

### Создание нового товара:
```bash
curl -X POST http://localhost:8000/items \
  -H "Content-Type: application/json" \
  -d '{"name": "Смартфон", "description": "Новый смартфон", "price": 699.99, "category": "Электроника"}'
```

### Поиск:
```bash
curl http://localhost:8000/search?q=ноутбук
```

## Документация API

FastAPI автоматически генерирует интерактивную документацию:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

## Структура проекта

- `main.py` - Основной файл приложения с настройкой сервера
- `http_handler.py` - Обработчики HTTP запросов
- `run_server.py` - Скрипт для удобного запуска сервера с автоперезагрузкой
- `requirements.txt` - Зависимости проекта
- `README.md` - Документация проекта
- `venv/` - Виртуальное окружение Python
